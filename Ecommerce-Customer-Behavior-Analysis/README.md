# Project Overview

This project analyzes customer purchasing behavior, engagement patterns, and discount strategies using an e-commerce transaction dataset containing 17,000+ orders.

The goal of the analysis is to identify key drivers of revenue, customer retention patterns, and the impact of discount strategies on purchasing behavior.

## Tools & Technologies:

- Python (Pandas, NumPy) – Data Cleaning
- Power BI – Dashboard & Visualization
- DAX – Analytical Measures
- GitHub – Project Documentation

## Key Business Questions

This analysis answers the following business questions:

1. Which customer segments generate the most revenue?
2. How does customer engagement affect purchase behavior?
3. What role do discounts play in driving revenue?
4. Which product categories and cities generate the highest revenue?
5. How does device type influence purchasing patterns?

## Dashboard Pages

1️⃣ Executive Overview
Provides a high-level summary of business performance including revenue, orders, customer retention, device usage, and category-level sales performance.

2️⃣ Customer Behavior Insights
Analyzes customer engagement patterns, age group performance, session duration, browsing behavior, and monthly revenue trends.

3️⃣ Pricing & Discount Analysis
Evaluates discount strategies, category-level pricing behavior, and the relationship between discounts and purchase quantity.
